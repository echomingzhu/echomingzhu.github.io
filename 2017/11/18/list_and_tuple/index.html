列表和元组

发表于 2017-11-18   |   分类于 python   |     |   阅读次数

列表和元组都是Python内建的序列，主要区别在于列表可以修改，而元组不可以。接下来我们先探讨下序列的通用操作：索引、分片、加、乘和检查某个元素是否属于序列。

序列通用操作

索引

序列中的所有元素都是有编号的，和C语言的数组下标一样，都是从0开始递增的。一种惯用法是字符串本身直接使用索引，而不是用一个变量引用他们。

分片

索引用来访问序列中的单个元素，分片操作可以用来访问序列一定范围内的子序列。分片通过冒号分开的两个索引实现。需要注意的是，第一个索引包含在提取的子序列中，第二个索引不包含在子序列中。

乘

用数字乘以一个序列会得到一个新的序列。

这里顺便提下None这个Pythn内建值，代表什么都没有，我们可以用该方法初始化一个长度为10，而元素什么都没有（实际上每一个元素都是None）的序列，注意和空序列[]区分。

检查某个成员是否在序列中In

为了检查某个值是否在序列中，可以使用In运算符。
